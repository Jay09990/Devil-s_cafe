# 👹 Devil's Cafe — The Den Never Sleeps

A premium, production-ready **Digital Hub** for Devil's Cafe, Motera. This isn't just a restaurant website; it's a high-performance, cyberpunk-infused digital identity designed to dominate the late-night economy of North Ahmedabad.

---

## ⚡ Core Philosophy
Devil's Cafe is built on a **Cyberpunk-Noir** aesthetic—dark, moody, and intense. It is engineered for speed, premium user experience, and absolute search engine dominance.

## 🛠️ Tech Stack
- **Frontend Framework**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) (The next generation of CSS utility-first architecture)
- **Animations**: [GSAP](https://gsap.com/) (ScrollTrigger, staggered entry, and glassmorphism motion)
- **Status Engine**: Custom React Hook for real-time "Open/Closed" logic (Handles 2 AM midnight crossover)
- **SEO**: JSON-LD Structured Data (SEOSchema), fully optimized Meta Tags, and dynamic Sitemaps.

---

## 🔥 Key Features

### 1. The Signature Lab
A minimalist, high-end menu system featuring over 75+ real items, categorized for speed and readability. Designed with a dotted-leader layout to evoke an artisanal, premium feel.

### 2. High-Performance SEO
- **Rich Snippets**: Integrated JSON-LD for Google Knowledge Panels and Rich Search Results.
- **Local SEO**: Geotagging and specific location optimization for Motera, Ahmedabad.
- **Performance**: High `fetchpriority` on hero assets, preconnected fonts, and lazy loading for scroll-depth performance.

### 3. Dynamic Status
A real-time logic engine that reflects the Cafe's late-night status (Open 9:00 AM – 2:00 AM), ensuring customers always know when the "Devil" is awake.

### 4. Interactive UX
- **Glassmorphism**: Modern, blurred UI elements for a sleek, industrial look.
- **Scroll-Triggered Motion**: Cinematic staggered entry for all major sections via GSAP.
- **WhatsApp Integration**: Direct, one-click ordering and inquiry channel.

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone [your-repo-link]
   ```

2. **Install dependencies**:
   ```bash
   cd client
   npm install --legacy-peer-deps
   ```

3. **Run the Development Server**:
   ```bash
   npm run dev
   ```

4. **Build for Production**:
   ```bash
   npm run build
   ```
